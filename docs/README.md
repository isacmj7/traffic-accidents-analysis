## How to Run

1. Install packages:
   ```
   pip install -r requirements.txt
   ```

2. Download dataset from OpenCity:
   https://data.opencity.in/dataset/road-accidents-in-india-2023

3. Place all CSV files in the `data/` folder (see data/README.md for file names)

4. Open notebook:
   ```
   jupyter notebook notebooks/01_traffic_accidents_analysis.ipynb
   ```
