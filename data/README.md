# Data Folder

Download the data files from:
https://data.opencity.in/dataset/road-accidents-in-india-2023

Required files:
- State-wise Road Accidents Data from 2019-2023 → save as `state_wise_accidents.csv`
- State-wise Road Fatalities Data from 2019-2023 → save as `state_wise_fatalities.csv`
- Road Accidents Classified by Type of Collision → save as `collision_types.csv`
- Road Accidents by Type of Violation 2023 → save as `violations.csv`
- Road Accidents with Absence of Safety Devices → save as `safety_devices.csv`
- Road Crash Fatalities by Road Users 2023 → save as `road_users_fatalities.csv`

Place all CSV files in this folder before running the notebook.
